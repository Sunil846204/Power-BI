# Power-BI
“Power BI practice projects, notes, and sample dashboards for data analysis and visualization.”
Comprehensive Uber Analytics dashboard built with Power BI, analyzing ride trends, city-wise performance, payment mode distribution, revenue insights, and driver ratings. Includes breakdowns by vehicle type (Auto, Mini, Sedan, SUV), city-level earnings, and customer payment preferences (Cash, Card, UPI, Wallet). Designed to provide actionable insights into operational efficiency, customer behavior, and revenue optimization across major Indian cities.
